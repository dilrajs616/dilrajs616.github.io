Making Demo Project

Initially I started working on this demo project as a fullstack developer. I used ejs as frontend and nodejs for backend. I nearly completed the project this way alone. It did not require a separate frontend framework like react. EJS was good enough. But then I decided that removing views from backend is a better idea because that way this server can also be used as backend for any Khalis app that may want to implement voice search feature. So with this in mind. I made two project. One is full stack web app with frontend done in ejs and backend in nodeJS. The link to that project is given below: 

https://github.com/KhalisFoundation/gurbani_search

Second project with pure backend and react front end is linked here Khalis Demo.
